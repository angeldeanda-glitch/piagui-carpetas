Conexión con Google Sheets
Cómo conectar paso a paso:
1. Abre tu Google Sheet → Extensiones → Apps Script
2. Borra todo el contenido y pega el código Apps Script que te daré abajo
3. Guarda (Ctrl+S) → Implementar → Nueva implementación
4. Tipo: Aplicación web · Ejecutar como: Yo (tu cuenta) · Acceso: Cualquier usuario
5. Haz clic en Implementar, autoriza los permisos y copia la URL
6. Pégala en el campo de arriba y guarda

Estructura que escribe el script en el Sheet:
A = CARPETA   B = KUNDEN   E = MODELO   F = ENTREGADO (checkbox)
